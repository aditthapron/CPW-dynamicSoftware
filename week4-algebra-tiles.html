📖 เรียนรู้แผ่นพีชคณิต

แผ่นพีชคณิต (Algebra Tiles) ช่วยให้เรามองเห็นภาพพหุนามได้ชัดเจนขึ้น!

ขนาดของแผ่น:
• 🟨 แผ่นเล็ก: กว้าง 1 ยาว 1 → พื้นที่ = 1
• 🟩 แผ่นยาว: กว้าง 1 ยาว x → พื้นที่ = x
• 🟦 แผ่นใหญ่: กว้าง x ยาว x → พื้นที่ = x²

🚀 สถานการณ์จริง

เลือกสถานการณ์ → กดจำลองลงกระดาน → กด “🧹 ลบหักล้าง” → นับแผ่นที่เหลือ แล้วพิมพ์คำตอบ

ตัวอย่างที่ 1 · ผังเมือง
มีที่ดินจัตุรัสด้าน x กม. จำนวน 2 แปลง (2x²) · รัฐเวนคืนสร้างทางด่วน x² จำนวน 1 แปลง (−x²) และถนนกว้าง 1 ยาว x จำนวน 2 เส้น (−2x)
เหลือพื้นที่เท่าไร?
ตัวอย่างที่ 2 · สวนผัก
มีแปลงผักจัตุรัสด้าน x เมตร จำนวน 3 แปลง (3x²) · ขุดบ่อน้ำจัตุรัส x เมตร 1 บ่อ (−x²) และทำทางเดินกว้าง 1 ยาว x จำนวน 3 ทาง (−3x)
เหลือพื้นที่ปลูกเท่าไร?
✏️ ตอบคำถาม

หลังกด 🧹 ลบหักล้าง แล้ว นับแผ่นที่เหลือ — เหลือพื้นที่เท่าไร?

✖️ การคูณพหุนามด้วยแบบจำลองพื้นที่

การคูณ (x + a)(x + b) ก็คือการหา พื้นที่ ของสี่เหลี่ยมผืนผ้าที่มีด้านกว้าง (x + a) และด้านยาว (x + b) — ปรับค่าด้านล่างแล้วนับแผ่นบนกระดานดูสิ!

💡 พื้นที่ทั้งหมด = (แผ่น 🟦 x²) + (แผ่น 🟩 x) + (แผ่น 🟨 หน่วย)
แผ่น x² มาจาก x·x, แผ่น x มาจาก x·1, และแผ่นหน่วยมาจาก 1·1
ด้านกว้าง = x + 2
ด้านยาว = x + 3
(x + 2)(x + 3) = x² + 5x + 6
🎯 ฝึกคูณพหุนาม (5 ข้อ)
สูตรลัด: (x + a)(x + b) = x² + (a + b)x + (a·b)
ค่าติดลบก็ใช้สูตรเดียวกัน เช่น (x − 2)(x + 3) = x² + x − 6
จงหาผลคูณ
