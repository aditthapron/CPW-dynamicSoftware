สมการเส้นตรง: y = mx + b
• m (ความชัน - Slope): บอกความชันและลาดเอียงของกราฟ
• b (จุดตัดแกน Y): ตำแหน่งจุดตัดบนแกนตั้ง (x = 0)

สมการกำลังสอง: y = ax² + bx + c
• a (ทิศทาง): a > 0 หงาย, a < 0 คว่ำ (ยิ่งไกลจากศูนย์ ยิ่งแคบ)
• b (แกน): เลื่อนตำแหน่งแกนสมมาตรซ้าย/ขวา
• c (จุดตัดแกน Y): จุดตัดแกนตั้ง (0, c)

การจัดรูปสมการกำลังสองจากกราฟ
เราสามารถหารูปแบบแยกตัวประกอบได้จากการสังเกตจุดตัดแกน X (รากของสมการ) สมมติคือ $x = r_1$ และ $x = r_2$ :
y = a(x - r₁)(x - r₂) *โดย $a$ คือสัมประสิทธิ์ระบุความคว่ำหรือหงาย
