ทำความเข้าใจพาราโบลา
y = a·x² + b·x + c

ขั้นที่ 1: ค้นพบสัมประสิทธิ์ a (คว่ำ vs หงาย) ⏳

สัมประสิทธิ์ a คุมรูปทรงของกราฟ ลองปรับสไลเดอร์ a ให้มีค่าติดลบ (< 0) เพื่อทำให้พาราโบลาคว่ำลงเป็นรูปโดม

ขั้นที่ 2: ค้นหาจุดตัดแกน Y ด้วยค่า c 🔒

ค่าคงที่ c บอกจุดที่ตัดแกนตั้ง Y (เมื่อ x = 0) ลองขยับสไลเดอร์ c ไปที่ 3.0

🧮 ลองแทนค่า x เพื่อหาพิกัด (x, y) บนกราฟ:

กำหนดให้ x =
y = 1.00·(0)² + 0.0·(0) + -3.0
y = 0.00 + 0.00 + (-3.0) = -3.00
จะได้พิกัดบนกราฟคือ: (0, -3.0)

❓ คำถาม: ถ้าสมการคือ y = x² - 2x - 5 กราฟจะตัดแกน Y ที่จุดใด?

ขั้นที่ 3: ค้นหาจุดยอดด้วย b 🔒

สัมประสิทธิ์ b ย้ายตำแหน่งแกนสมมาตร ลองตั้งค่า a = 1.00, c = 0.0 แล้วปรับ b = -4.0 เพื่อค้นหาจุดยอด
